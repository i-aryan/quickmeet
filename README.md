
# SignCall
A video chat/meeting webapp using WebRTC and WebSockets (socket.io). Basically a Google Meet clone + a collaborative whiteboard.


## Features

- Video Calls
- Chatbox
- Collaborative Whiteboard
- Screensharing
- Responsive design
